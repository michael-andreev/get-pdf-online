Technologies

Here is the main technologies that we are using in our tool:

Microsoft .NET Core
- a blazing fast, lightweight and modular platform for creating web applications and services that run on Windows, Linux and Mac. It's fully open source.
Angular
- an open source platform for building mobile and desktop web applications led by the Angular Team at Google and by a community of individuals and corporations.
Node.js
- a JavaScript runtime built on Google's Chrome V8 high-performance, open source JavaScript engine.
LibreOffice
- the most powerful Free and Open Source office suite on the market.
SQLite
- a self-contained, high-reliability, embedded, full-featured, public-domain, SQL database engine. SQLite is the most used database engine in the world.
PrimeNG
- a collection of rich UI components for Angular. PrimeNG is a sibling of the popular JavaServer Faces Component Suite, PrimeFaces. All widgets are open source and free to use under MIT License.
Bootstrap
- the most popular HTML, CSS, and JS framework for developing responsive, mobile first projects on the web.
Swagger
- the world’s largest framework of API developer tools for the OpenAPI Specification(OAS), enabling development across the entire API lifecycle, from design and documentation, to test and deployment.
Microsoft WCF
- Microsoft’s unified programming model for building service-oriented applications. It enables developers to build secure, reliable, transacted solutions that integrate across platforms, and interoperate with existing investments.
Magick.NET
- a powerful image manipulation library that supports over 100 major file formats (not including sub-formats).
WIX toolset
- the most powerful set of tools available to create Windows installation packages.
